# SAE_3.01

Application de navigations vers les différents parking de Metz disponible dans notre BDD.
Possible de trouver le parking le plus proche en 1 clique !

---

**Application**

Structure :
- HTML / JS / CSS => FrontEnd
- PHP => BackEnd
- Cordova => Version Android
- PHPmyAdmin => Notre BDD

**COMMENT L'utiliser**

1 - Se connecter au VPN de l'IUT
2 - Accéder à l'URL pour tester la version web :
  devweb.iutmetz.univ-lorraine.fr/~e58632u/sae3/src/
3 - Ou, installer et télécharger l'APK (disponible dans l'espace release du REPO) sur un téléphone android
